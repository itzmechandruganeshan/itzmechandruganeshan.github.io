### Final Year Project / Thesis

**"AI-Based Framework for Automated Pathogen Classification via Real-Time PCR Data Analysis"**

**Problem Solved:** Manual interpretation of rt-PCR (real-time Polymerase Chain Reaction) diagnostic data by clinicians is laborious, time-consuming, and highly prone to human variability. My project aimed to automate and standardize this process using machine learning.

**Approach & Methodology:**
- Applied advanced signal processing techniques — **Savitzky-Golay smoothing** and **B-Spline interpolation** — to clean and normalize raw PCR curve data before feeding it into models.
- Extracted critical diagnostic features from two types of data: **High-Resolution Melting (HRM) curves** and **Amplification Curves**.
- Built a classification system using the **Random Forest algorithm** to identify and classify pathogens accurately from these features.

**Three Core Components Built:**
| Component | Purpose |
|---|---|
| **REXTRACTOR** | Raw data acquisition and preprocessing pipeline |
| **PyMLRS** | Feature engineering and machine learning model training |
| **Pathogen Detector** | Predictive result interpretation and diagnostic output |

**Technologies Used:** Python, Random Forest (Scikit-learn), Savitzky-Golay Filter, B-Spline Interpolation, Signal Processing libraries

**Impact:** The system provided a faster, more accurate, and standardized molecular assay interpretation tool — reducing clinician workload and improving diagnostic consistency.

> This project was deeply interdisciplinary, sitting at the intersection of bioinformatics, signal processing, and machine learning — and it taught me how to apply AI to high-stakes, real-world domains where accuracy is non-negotiable.
