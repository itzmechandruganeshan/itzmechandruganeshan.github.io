# Digital Chandru - Behavioral Story (Proudest Moment)

## 1. The Achievement
The project I am most proud of is PyMLRS, an automated machine learning framework I built to diagnose clinical pathogens from real-time PCR data. I built the entire pipeline from scratch—from writing a custom XML parser for raw fluorescence data, to applying signal processing filters (Savitzky-Golay) for noise reduction, and finally training the ML classification model. I ultimately published it as an open-source library on PyPI.

## 2. Why it Matters
This project sits at the intersection of bioinformatics, signal processing, and machine learning. It wasn't just another CRUD app or API wrapper—it solved a highly complex, critical-care bottleneck. By automating the extraction and interpretation of High-Resolution Melt (HRM) and Amplification curves, the system eliminates human error and drastically reduces the manual workload for clinicians diagnosing the Meningoencephalitis Panel. It proved that I can take a deep, unstructured domain problem and architect a robust, deterministic software solution.
